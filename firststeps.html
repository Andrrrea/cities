First steps with Highcharts

The first steps explain how to use highcharts in Javascript and show how to integrate external data sources.

Highcharts einbinden

1. Binden Sie die highcharts library in dem HTML Dokument Ihrer Webpage ein

<script src="https://code.highcharts.com/highcharts.js">

Erstellung eines Balkendiagramms

2. Fügen Sie ein div in Ihre Webseite ein. Geben Sie ihm eine ID und legen Sie eine bestimmte Breite und Höhe fest, die die Breite und Höhe Ihres Diagramms sein werden.

<div id="container_beispiel" style="width:100%; height:400px;"></div>

3. Ein Diagramm wird initialisiert, indem das JavaScript-Tag <script> </script> an beliebiger Stelle in einer Webseite eingefügt wird, das den folgenden Code enthält. Das div aus #1 wird darein referenziert.

document.addEventListener('DOMContentLoaded', function () {
const chart = Highcharts.chart('container_beispiel', {
chart: {

},

Externe Datenquellen einbinden

6. Externe Datenquellen wie CSV, XML und JSON können in Highcharts verwendet werden. Während XML und CSV Daten noch ein Preprocessing durchlaufen müssen, lassen sich JSON Dateien (da sie schon im Javascript Format sind) direkt ohne Preprocessing einbinden. Da JSON eine häufige Datasource in der Praxis ist, soll hier die Einbindung einer JSON Datei erläutert werden.

7. JSON Dateien werden mittels AJAX in eine Javascript Datei eingebunden. AJAX (Asynchronous JavaScript and XML) ist eine Methode, die speziell für die Einbindung externer Inhalte in Javascript entwickelt wurde.
Achtung: AJAX als Methode funktioniert jedoch nur, wenn die externen Dateien auf einem Server liegen. Auf lokal gespeicherte Dateien auf dem Computer kann nicht zugegriffen werden. Dies ist aus Sicherheitsgründen nicht erlaubt, um nur Inhalte anzeigen lassen zu können, die auch online stehen (sollen).
Das verwendete Object, welches die erlaubt, externe Daten in Javascript einzubinden heißt XMLHttpRequest

JSON in Javascript einbinden

window.onload = AJAX_data_cities; // Name der Funktion
//alert("Skript funktioniert");

function AJAX_data_cities()
{
var req = new XMLHttpRequest();// Aufrufen des Objekts zum Einbinden von JSON
req.onreadystatechange = function ()
{

if (req.readyState == 4)
{
if (req.status == 200 || req.status == 0)
{
var res = req.responseText;
var jobj = JSON.parse(res);
if(jobj) {
Verarbeiten(jobj); // Funktion Verarbeiten ist weiter unten definiert --> NOTWENDIG
}
}
else alert("Error" + req.statusText);
}
}
req.open("GET", "data_cities.json", true); // relativen Pfad und Name der JSON anpassen
req.send();
}


// Funktion Verarbeiten für die Nutzung der JSON Daten zum Zugriff auf die einzelnen Objekte
function Verarbeiten(jobj)
{
var cities = jobj.cities;

var city_data = [];
for (var i in cities) {
var jdata = {
Name: cities[i].name,
Jahr: cities[i].Jahr,
Einwohnerzahl: cities[i].Einwohnerzahl,
Groesse: cities[i].Groesse
}
cities.push(jdata);

}
